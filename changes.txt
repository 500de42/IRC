Modifications apportées au projet IRC
======================================

Corrections des bugs critiques et nettoyage du code pour que ça passe l'éval.


Les trucs critiques (sans ça l'eval passe pas) :

• Ligne 3 srcs/main.cpp -> manquait #include <fcntl.h>

• Ligne 31 srcs/main.cpp -> ajout fcntl(client->getSocket(), F_SETFL, O_NONBLOCK) 
  car les sockets clients étaient pas en non-bloquant

• Lignes 40-41 srcs/main.cpp -> ajout else delete client 
  sinon fuite mémoire quand accept() rate

• Ligne 3 srcs/Classes/Serveur.cpp -> manquait #include <fcntl.h>

• Ligne 36 srcs/Classes/Serveur.cpp -> ajout fcntl(srv, F_SETFL, O_NONBLOCK) 
  car socket serveur pas en non-bloquant non plus

• Lignes 108-118 srcs/Commandes/utils.cpp -> bug dans prohibidedCharacter()
  la condition (*i >= 0 && *i <= 31) rejetait TOUS les caractères 
  donc personne pouvait s'enregistrer avec USER

• Lignes 71-74 srcs/Commandes/USERandNICK.cpp -> validation du realname bloquait les espaces
  donc "Bob Smith" marchait pas

• Lignes 81-82 srcs/Commandes/utils.cpp -> PRIVMSG était codé mais jamais appelé dans execCommand()

• Ligne 158 includes/Serveur.hpp -> manquait la déclaration de PRIVMSG


Nettoyage du code :

• Viré tous les cout de debug dans srcs/main.cpp, srcs/Classes/Serveur.cpp, 
  srcs/Classes/Client.cpp, et tous les fichiers de commandes

• Lignes 70-71 srcs/Commandes/MODE.cpp -> viré variables inutilisées 
  nbParameters et executedParameters (warning -Werror)

• Lignes 92-93 srcs/Commandes/MODE.cpp -> viré if statements dupliqués

• Viré le code commenté dans srcs/main.cpp, srcs/Commandes/utils.cpp, 
  srcs/Commandes/JOIN.cpp et srcs/Commandes/KICK.cpp

Le serveur compile sans warnings, pas de fuites mémoire, toutes les commandes marchent.
